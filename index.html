My Projects
Language Translation Model

The language translator tool was built using NLP (Natural Language Processing), machine learning, and basic concepts of deep learning. It leveraged transformers, Hugging Face, and dataset libraries. Through effective data cleaning and management, a robust language translator was developed, capable of translating English to Hindi.

Sentimental Analysis

The Sentiment Analysis project was designed to analyze and understand the sentiments expressed in textual data. This project leverages Natural Language Processing (NLP) techniques and machine learning algorithms to determine the emotional tone of a given piece of text. The primary goal is to classify the sentiment of the text into categories such as positive, negative, or neutral.

Blog Creator

Streamlining the blog generation process has been made remarkably simple with minimal inputs, resulting in the creation of bespoke and engaging content tailored to a wide range of job profiles. This efficiency is achieved by harnessing essential libraries like Streamlit and Lang chain, facilitating seamless development. Leveraging the formidable capabilities of the Llama 2 language model further enhances this process. By integrating the Llama 2 model through the Transformers class, which is pre-trained on extensive datasets, we ensure the production of insightful blog content with ease.

Movie Recommend System

Utilizing Python, particularly Streamlit, I crafted a personalized movie recommendation platform that interfaces seamlessly with the TMDB API for comprehensive movie data. This endeavor involved leveraging essential Python libraries like Pickle for streamlined data storage and Requests for efficient API calls, ensuring a smooth and responsive system for users to navigate. The platform boasts key features such as Personalized Picks, tailored to individual user preferences, and a sleek interface designed to enhance overall user experience and engagement.

Plot Prediction

The Plot Price Prediction project aims to forecast the prices of plots in Bangalore using machine learning techniques. By leveraging historical data and relevant features, this project aims to create a predictive model that estimates the price of real estate plots in Bangalore.

OLlama Model

I developed a web application powered by a custom OLlama model, enabling users to input questions and queries and receive answers akin to those generated by OpenAI's model.
